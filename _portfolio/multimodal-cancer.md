## Multi-modal Cancer Analysis

This project explores the integration of multiple data modalities — histopathology images, transcriptomic profiles, and clinical metadata — to build more comprehensive models of cancer biology.

### Key Areas

- **Image-omics fusion** — Combining visual features from WSIs with gene expression data
- **Survival prediction** — Multi-modal models for patient outcome prediction
- **Pathway analysis** — Linking visual phenotypes to molecular pathways

### Approach

We use cross-attention mechanisms and multimodal transformers to learn joint representations across data types, enabling discoveries that single-modality analyses might miss.
